silicate

A well-set book page, not a landing page.

One stylesheet. Styles raw HTML. No frameworks, no classes, no build step. The browser's defaults are ninety percent right; we fix line length, leading, and contrast — and stop.

Use it

<link rel="stylesheet" href="silicate.css">

That is the entire integration. Now write HTML. If a tag exists for the job, use the tag.

Theme it

Override --silicate-* tokens at :root. Five lines is enough.

:root {
  --silicate-font-body: "EB Garamond", Georgia, serif;
  --silicate-accent: #458588;   /* gruvbox blue */
  --silicate-bg: #f9f5d7;
  --silicate-fg: #3c3836;
  --silicate-measure: 38rem;
}

Ground rules

  1. Use the tag. If HTML has it, use it. No <div class="card">.
  2. No classes. If you find yourself writing one, you are in the wrong system.
  3. No new colors. The accent is the accent.
  4. No shadows, no radius, no gradients. Ever.
  5. One column, ~70ch. Escape with <figure data-bleed> only when the content demands it.
  6. Words over icons. A label is clearer than a glyph.
  7. Sentence case. No Title Case in headings.
  8. Light/dark with the OS. No toggle.
