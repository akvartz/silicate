silicate

A minimalist design system. One stylesheet. No classes. No build step. Style raw HTML and let the content do the work.

This page is the showcase, the docs, and the test bed — all at once.

Philosophy

silicate descends from motherfuckingwebsite.com, bettermotherfuckingwebsite.com, and thebestmotherfucking.website. The original premise still holds: the browser already knows how to render text. We give it just enough help — readable measure, comfortable leading, honest contrast — and step out of the way.

  • Content first. Markup carries meaning; CSS only refines it.
  • Style elements, not classes. If a tag exists for the job, use it.
  • One file. Ship the whole system in a single stylesheet.
  • Tokens, not magic numbers. Every value is a CSS custom property you can override.
  • Light + dark, by default. Respect prefers-color-scheme.
  • Accessible by default. Visible focus, generous tap targets, real contrast.

Use it

<link rel="stylesheet" href="silicate.css">

Then write HTML. That's the whole API.

Tokens

Every design decision is exposed as a CSS custom property under the --silicate-* namespace. Override at :root to retheme.

The token surface
GroupTokens
Type--silicate-font-sans, --silicate-font-mono, --silicate-font-size, --silicate-line-height, --silicate-measure
Space--silicate-space-1 through --silicate-space-7

Retheming example

:root {
  --silicate-measure: 42rem;
  --silicate-accent: #c2410c;
  --silicate-font-sans: "Inter", system-ui, sans-serif;
}

A paragraph of body copy. The default measure is 38rem — about seventy characters per line, which keeps the eye from getting lost on its way back. Body size is 1.125rem with a 1.6 line height. What's in a silicate?

Silicates are minerals built from silicon-oxygen tetrahedra. They form the bulk of Earth's crust. Quartz is one. Mica is another.
