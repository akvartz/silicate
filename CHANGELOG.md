# Changelog

## Unreleased

### Added

- **Foundation.** Token-driven single-file stylesheet that styles raw HTML.
- **Tokens.** Type, space, and color scales as `--silicate-*` custom
  properties. Light + dark via `prefers-color-scheme`.
- **Element coverage.** Headings, body text, links, lists, code, blockquote,
  tables, figures, forms, disclosure, media, rules.
- **Print styles.** Page reflow, neutralized link color, surface backgrounds
  for code.
- **Showcase.** `index.html` exercises every styled element and doubles as
  the living style guide.
- **Docs.**
  - `README.md` — philosophy, principles, install, token surface.
  - `docs/elements.md` — per-element behavior reference.
  - `docs/cookbook.md` — recipes for common patterns without utility classes.
